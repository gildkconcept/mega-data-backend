# Mega-data Backend - Déploiement Render

## Variables d'environnement requises

| Variable | Description | Exemple |
|----------|-------------|---------|
| `PORT` | Port d'écoute | `10000` |
| `JWT_SECRET` | Clé secrète JWT | `votre_clé_secrète` |
| `FRONTEND_URL` | URL du frontend | `https://...` |
| `NODE_ENV` | Environnement | `production` |

## Démarrage local
```bash
npm install
